Hosted endpoints

Install script shortcuts for GitHub Pages.

The deployment workflow copies the repository install scripts into the final Pages artifact so these URLs stay stable and easy to document.

Linux / macOS

Hosted shell installer

curl -fsSL https://christianhelle.github.io/argiope/install | sh

Download `install` directly

Windows

Hosted PowerShell installer

irm https://christianhelle.github.io/argiope/install.ps1 | iex

Download `install.ps1` directly

What the scripts do: fetch the latest GitHub release, pick the matching platform artifact, install the binary to a practical location, and leave you with a command-ready `argiope` executable.
